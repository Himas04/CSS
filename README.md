# CSS Practice Exercises

This repository contains a collection of HTML and CSS practice files and tasks organized by selector type and exercises. It's intended for learning and experimenting with CSS selectors and pseudo-classes/pseudo-elements.

## How to use

- Open the HTML files in your web browser (double-click or use a local server).
- Most pages reference `styles.css` files found either in the root or in subfolders.

## Structure

- **index.html**: Main demo page.
- **index2.html**: Alternate demo page.
- **styles.css**: Global styles for root demos.
- **task.html**: General task page.
- **Attribute Selector/**: Exercises focused on attribute selectors (files `task1-11.html` through `task20.html` and a folder-specific `styles.css`).
- **Pesudo Selector/**: Exercises for pseudo-classes and pseudo-elements (includes `navigationmenu.html`, `classslector.html`, `elementselector.html`, and `styles.css`).
- **Selector/**: General selector exercises (`selectorTask1-20.html`, `que21.html`–`que25.html`).

## Tips

- Use the browser DevTools (Inspect) to view applied CSS rules and experiment live.
- If styles don't appear, check that the HTML file links to the correct `styles.css` path.
